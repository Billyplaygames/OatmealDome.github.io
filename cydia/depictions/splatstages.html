Description

  • SplatStages is an app that tells you the current and future Splatoon map rotations. It can also display information about upcoming, currently ongoing, and past Splatfests.

    The source code for this app is available on GitHub! Please send any bug reports or feature requests there.

  • This app contains MBProgressHUD by matej and NSAttributedString+CCLFormat by Cocode Ltd. The Project Paintball font was created by JapanYoshi/theFIZZYnator. Schedule data is provided by splatoon.ink, and I provide the Splatfest data. The settings icon was made by icons8.com.
